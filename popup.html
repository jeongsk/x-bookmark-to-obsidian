북마크 페이지 동기화

먼저 X 북마크 페이지를 연 다음, 이번 회차의 목표 동기화 개수를 설정하세요. 스크롤 깊이는 플러그인이 자동으로 추정합니다.

실행 중에는 X 북마크 페이지 우측 상단에 상시 진행 패널이 표시됩니다.

최근 일괄 동기화: 기록 없음.

이번 회차 동기화 완료 후, 성공한 북마크를 삭제할 수 있습니다.

실수 방지를 위해 기본값은 자동 삭제 안 함입니다.

설치 안내

  1. chrome://extensions에서 개발자 모드를 켜세요.
  2. 이 확장 프로그램 디렉토리를 로드하세요.
  3. 디렉토리 내 install.command를 실행하세요.
  4. Chrome을 재시작하세요.
